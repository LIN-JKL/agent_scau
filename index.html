校园智慧信息网络中心问答助手

项目简介

这是一个校园智慧信息网络中心问答助手系统，提供了基于网页的用户界面和后端服务，可以帮助师生解答关于校园网络、信息化服务等方面的问题。

重要提示：这是一个需要在本地运行Python服务器的应用程序，而不是一个可以直接通过GitHub Pages访问的静态网站。请按照以下说明在本地运行此应用。

快速开始

前提条件

  • 安装Python 3.x
  • 具备基本的网络环境

启动服务器

1. 克隆此仓库到本地：

git clone [仓库地址]

2. 导航到项目根目录：

cd 校园智慧信息网络中心问答助手

3. 运行以下命令启动静态文件服务器（有两种选择）：

选择1：简单服务器（使用8081端口）

python simple_server.py

选择2：更完善的服务器版本（推荐，使用8082端口）

python final_server.py

4. 服务器启动后，在浏览器中访问以下地址：

  • 简单服务器：http://localhost:8081/
  • 完善服务器：http://localhost:8082/

5. 系统会自动重定向到完整问答助手界面
